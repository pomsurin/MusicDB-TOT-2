# MusicDB-TOT-2
## Please read the below contribution.

### Data format
{


"id": string, # id > 0 and id is natural number


"music_name": string,


"music_genre": string,


"artist": string,


"year": "string,


}

<hr>

### Example,
{ <br>
"id": "0", <br>
"music_name": "Melon Juice", <br>
"music_genre": "Pop", <br>
"artist": "CGM48, BNK48", <br>
"year": "2020", <br>
} <br>
